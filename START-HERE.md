# 🚀 CCBoss - Start in 1 Step!

## Step 1: Install & Run

```bash
cd ~/CC/ccboss && ./ONE-STEP-INSTALL.sh
```

That's it! CCBoss is now installed.

## Step 2: Get an API Key (Pick One)

**Option A: SkillBoss** (Easiest - 100+ models with 1 key)
Visit: https://skillboss.co/console
Copy your API key, then run:
```bash
cb setup
```

**Option B: Use Kimi** (Free for Chinese users)
Visit: https://platform.moonshot.cn
Get your API key, then run:
```bash
cb setup
```

## Step 3: Start Using

```bash
cb                    # Interactive chat
cb ask "hello"        # Quick question
```

## That's it! 🎉

**All commands:**
- `cb` or `ccb` or `ccboss` or `boss` - All work!
- `cb ask "question"` - Quick one-shot question
- `cb setup` - Configuration wizard
- `cb models` - List available models
- `cb --help` - Show all commands

**Shortest command in the world: just 2 characters!**
```bash
cb
```
