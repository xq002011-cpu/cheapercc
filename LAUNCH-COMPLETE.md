## 🚀 How Users Install (No npm needed!)

```bash
# Clone the repo
git clone https://github.com/xq002011-cpu/cheapercc.git
cd cheapercc

# Install dependencies
npm install

# Link globally
npm link

# Setup
cc setup

# Use
cc ask "Hello!"
```

**That's it!** Works instantly from GitHub.
